## 📖 使用示例

### 快速开始
```bash
cd patterns

# 编译所有测试
./build_all_tests.sh

# 运行所有测试
./run_all_tests.sh
```

### 编译特定模式
```bash
# 编译单例模式测试
./build_all_tests.sh creational/singleton

# 运行单例测试
./build/singleton_test_basic
./build/singleton_test_thread_safety
```

### 手动编译
```bash
# 基础测试
g++ -std=c++14 -I../include \
    creational/singleton/test_basic.cpp \
    -o test_singleton

# 线程测试
g++ -std=c++14 -I../include -pthread \
    creational/singleton/test_thread_safety.cpp \
    -o test_thread
```
